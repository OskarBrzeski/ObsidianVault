# Speed Classes
## Big C with number inside
Indicates minimum sequential write speed in MB/s.
2, 4, 6, 10
## Bucket with number inside
Indicates minimum sequential write speed is 10MB/s or 30MB/s.
1, 3
## V followed by number
Indicates minimum sequential write speed in MB/s.
6, 10, 30, 60, 90
## A followed by number
Indicates minimum read/write IOPS.
A1 is 1500 read / 500 write IOPS.
A2 is 4000 read / 2000 write IOPS.
