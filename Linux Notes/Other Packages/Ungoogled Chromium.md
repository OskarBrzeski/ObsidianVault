Chromium with Google parts stripped out.
https://github.com/ungoogled-software/ungoogled-chromium
# Installation
## CachyOS repositories
```bash
pacman -S ungoogled-chromium-bin
```
## Arch
```bash
paru -S ungoogled-chromium-bin
```
## Flatpak
```bash
flatpak install flathub io.github.ungoogled_software.ungoogled_chromium
```

# Configuration
## Extension store
Since this browser does not include the parts necessary to enable extension support, the following must be downloaded first: https://github.com/NeverDecaf/chromium-web-store/releases.
